Data Science Projects

2024 · Group Project · University of Toronto

CPI vs S&P 500 — Statistical Relationship Analysis

Investigated whether Consumer Price Index (CPI) and S&P 500 performance can predict each other using 20 years of monthly data (2000–2023). Resolved data quality issues including date inconsistencies and format mismatches, then produced dual-axis visualisations revealing a Pearson correlation of r = 0.84. Applied ADF stationarity tests and Granger causality — finding that S&P 500 significantly Granger-causes CPI (p < 0.01), but not vice versa.

Python Pandas NumPy Matplotlib SciPy Statsmodels Time Series Granger Causality
2024 · Group Project · University of Toronto

Japanese Cherry Blossom ML Prediction Model

Collaborated in a team of five to analyse 69 years of bloom data across 58 sites in all 47 prefectures of Japan (1955–2023). Cleaned 1.4M+ temperature values and merged multiple datasets. Implemented and compared 9 ML models including Gradient Boosting, Random Forest, and Elastic Net (lowest MSE: 0.10). Confirmed +2.7°C temperature rise over 50 years driving earlier bloom dates, with the most dramatic shifts post-1997.

Python Scikit-learn Pandas Seaborn Gradient Boosting Random Forest GridSearchCV Ensemble Methods
2026 · University of London Certification

Banknote Authentication via K-Means Clustering

Built an unsupervised ML model to classify genuine vs. forged banknotes on a dataset of 1,372 samples (OpenML), using wavelet-transform-derived features. Applied StandardScaler normalisation and progressively expanded from 2 to 4 features. Visualised cluster separation with Matplotlib scatter plots and critically evaluated the model — proposing supervised alternatives (logistic regression, random forests, neural networks) for improved accuracy.

Python K-Means Scikit-learn Unsupervised Learning Feature Engineering Matplotlib
2025 · Mini Project · University of Toronto

Parkinson's Disease Detection — XGBoost

Built a binary classification model to detect Parkinson's disease from 195 biomedical voice recordings across 31 patients (23 with PD). Extracted 22 dysphonia features, applied MinMaxScaler normalisation to the range [-1, 1], and trained an XGBoost classifier with an 80/20 stratified split. Evaluated model performance using accuracy_score, achieving strong discrimination between healthy and PD-positive cases.

Python XGBoost Scikit-learn MinMaxScaler Binary Classification Pandas NumPy
2025 · Mini Project · University of Toronto

Iris Flower Classification — SVM

Built a supervised classification model to identify three iris species (Setosa, Versicolour, Virginica) from petal and sepal measurements. Performed EDA with Seaborn pair plots and Matplotlib bar charts to visualise per-class feature averages. Trained a Support Vector Classifier with an 80/20 train-test split, evaluated with sklearn accuracy_score, and serialised the final model using Pickle for reuse and deployment.

Python Scikit-learn SVM Seaborn Matplotlib Pandas Pickle
